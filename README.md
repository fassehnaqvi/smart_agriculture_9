# ICT for Smart Agriculture: Drone Spraying & Resource Optimizer

## Mechanical Engineering ICT Semester Project

This project demonstrates how Information and Communication Technology (ICT) can improve modern agriculture using drone spraying systems and resource optimization techniques.

### Features

- Responsive Single Page Website
- Smart Agriculture Dashboard
- Drone Spraying System
- Resource Optimizer Calculator
- Glassmorphism Design
- Animated Sections
- Dark Mode Toggle
- Weather Widget Demo
- Mobile Friendly Design
- GitHub Pages Compatible

### Technologies Used

- HTML5
- CSS3
- JavaScript
